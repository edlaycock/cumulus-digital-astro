# Work screenshots

Drop client site screenshots here (PNG or JPG, ~1600px wide, showing the top of the homepage).

To use one, add this line to the matching entry's frontmatter in `src/content/work/<slug>.md`:

    image: ../../assets/work/<slug>.png

The browser-frame preview on the work cards and case study page picks it up automatically.
Suggested filenames: castellum.png, assetq.png, the-queen-stage.png, mew-developments.png,
oakbridge-group.png, electric-mx.png, surrey-hills-surfacing.png
